# 项目结构

一个良好的项目结构对于高效的部署至关重要。以下是推荐的项目结构：

```
project-root/
├── .env                   # 环境变量文件（服务器登陆密码等）
├── backup/                # 配置备份文件夹
│   ├── .env               # 备份 .env
│   ├── config.mjs         # 备份 config.mjs
├── config/                # 部署配置文件夹
│   ├── config.mjs         # 部署配置文件
├── logs/                  # 部署日志文件夹
│   ├── 2020-08-08/        # 年月日日志文件夹
│       ├── wukong.log     # 日志
```

## 最佳实践

1. **使用环境变量**
   - 将敏感信息（服务器密码）存储在 `.env` 文件中
